# نظام إدارة الطلبات - متجر البخور والعطور

نظام إدارة طلبات متكامل ومتطور لمتجر البخور والعطور، مبني باستخدام React و TypeScript مع قاعدة بيانات Supabase.

## المميزات الرئيسية

### 📋 إدارة الطلبات
- عرض جميع الطلبات مع إمكانية الفلترة المتقدمة
- تتبع حالة الطلبات (في الانتظار، مؤكد، قيد المعالجة، تم الشحن، تم التسليم، ملغي)
- إحصائيات شاملة للطلبات والإيرادات

### 🛍️ إدارة المنتجات
- إضافة وتعديل المنتجات
- تتبع المخزون
- تصنيف المنتجات
- رفع صور المنتجات

### 👥 إدارة العملاء
- قاعدة بيانات العملاء
- معلومات الاتصال والعناوين
- تاريخ الطلبات لكل عميل

### 🎫 نظام الخصومات
- خصومات مئوية وثابتة
- أكواد خصم قابلة لإعادة الاستخدام
- تتبع فعالية الخصومات
- **تحديد متعدد للكوبونات** مع عمليات مجمعة (حذف، تفعيل، إلغاء تفعيل)

### 📄 الفواتير والاستيكرز
- فواتير إلكترونية PDF
- استيكرز شحن احترافية
- طباعة وتحميل الفواتير

### 📥📤 الاستيراد والتصدير
- **تصدير الطلبات**: تصدير جميع الطلبات بصيغة CSV
- **استيراد الطلبات**: استيراد طلبات جديدة من ملف CSV
- **تصدير المنتجات**: تصدير قائمة المنتجات مع التفاصيل
- **استيراد المنتجات**: استيراد منتجات جديدة من ملف CSV
- **تصدير التصنيفات**: تصدير هيكل التصنيفات
- **استيراد التصنيفات**: استيراد تصنيفات جديدة

## التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - لغة البرمجة مع دعم الأنواع
- **Vite** - أداة البناء السريعة
- **Tailwind CSS** - إطار عمل CSS
- **React Router** - إدارة التنقل
- **React Hook Form** - إدارة النماذج
- **Zod** - التحقق من صحة البيانات
- **React Query** - إدارة حالة الخادم
- **React Hot Toast** - إشعارات المستخدم
- **Lucide React** - مكتبة الأيقونات

### Backend & Database
- **Supabase** - قاعدة البيانات والخدمات الخلفية
- **PostgreSQL** - قاعدة البيانات
- **Row Level Security** - أمان البيانات

### Utilities
- **date-fns** - معالجة التواريخ
- **jsPDF** - إنشاء ملفات PDF
- **react-to-print** - طباعة المكونات

## التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone <repository-url>
cd orders_sys
```

### 2. تثبيت التبعيات
```bash
npm install
```

### 3. إعداد متغيرات البيئة
```bash
cp env.example .env
```

قم بتحديث ملف `.env` بالقيم الصحيحة:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_APP_NAME=نظام إدارة الطلبات
VITE_APP_VERSION=1.0.0
```

### 4. إعداد قاعدة البيانات
1. أنشئ مشروع جديد في Supabase
2. انسخ محتوى ملف `database-schema.sql`
3. نفذ الاستعلامات في SQL Editor في Supabase

### 5. تشغيل المشروع
```bash
npm run dev
```

سيفتح المشروع على `http://localhost:3000`
