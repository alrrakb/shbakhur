# سجل التغييرات

جميع التغييرات المهمة في هذا المشروع ستُوثق في هذا الملف.

التنسيق مبني على [Keep a Changelog](https://keepachangelog.com/ar/1.0.0/)،
وهذا المشروع يتبع [Semantic Versioning](https://semver.org/lang/ar/).

## [غير مطلق] - 2024-01-XX

### أضيف
- نظام التحديد المتعدد للكوبونات مع العمليات المجمعة
- إمكانية حذف وتفعيل وإلغاء تفعيل عدة كوبونات دفعة واحدة
- واجهة مستخدم محسنة لأزرار التحديد المتعدد
- رسائل تأكيد واضحة للعمليات المجمعة
- دعم API للعمليات المجمعة على الكوبونات

### محسن
- تجربة المستخدم في صفحة كوبونات الخصم
- كفاءة إدارة الكوبونات
- اتساق واجهة المستخدم مع باقي الصفحات

### إصلاح
- مشاكل في عرض أزرار التحديد المتعدد
- تحسين الأداء عند التعامل مع عدد كبير من الكوبونات

## [1.0.0] - 2024-01-XX

### أضيف
- نظام إدارة الطلبات الكامل
- إدارة المنتجات مع التصنيفات المتعددة
- إدارة العملاء
- نظام الخصومات والكوبونات
- الفواتير الإلكترونية PDF
- استيكرز الشحن
- التحديد المتعدد للمنتجات والتصنيفات
- سحب وإفلات التصنيفات
- الاستيراد والتصدير (CSV)
- التصميم المتجاوب لجميع الأجهزة
- نظام المصادقة والأمان
- لوحة تحكم شاملة مع الإحصائيات

### التقنيات المستخدمة
- React 18 + TypeScript
- Vite للبناء
- Tailwind CSS للتصميم
- Supabase للقاعدة البيانات
- React Query لإدارة البيانات
- React Hook Form + Zod للنماذج
- jsPDF للفواتير
- Lucide React للأيقونات

## تنسيق السجل

### أنواع التغييرات
- **أضيف** للميزات الجديدة
- **غير** للتغييرات في الوظائف الموجودة
- **محسن** للتغييرات التي لا تؤثر على الوظائف
- **إصلاح** لإصلاح المشاكل
- **إزالة** للميزات المحذوفة
- **أمان** للتحديثات الأمنية

### الإصدارات
- **غير مطلق** للتغييرات غير المطلقة
- **[1.0.0]** للإصدارات المطلقة
- **[1.0.0-beta.1]** للإصدارات التجريبية
